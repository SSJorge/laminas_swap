Política de privacidad

Última actualización: 23 de Junio de 2026

Esta Política de privacidad explica cómo TruequeGol recopila, usa, almacena y protege la información de sus usuarios.

TruequeGol es una aplicación independiente para ayudar a personas a registrar láminas o tarjetas coleccionables, encontrar posibles intercambios y contactar a otros usuarios solo cuando exista interés mutuo.

Esta aplicación no está afiliada, patrocinada ni respaldada por marcas, editoriales, federaciones, eventos deportivos, álbumes oficiales ni empresas relacionadas con colecciones oficiales.

2. Datos que recopilamos

Podemos recopilar y almacenar los siguientes datos:

  • Correo electrónico.
  • Nombre de usuario.
  • Región y comuna seleccionada por el usuario.
  • Descripción escrita por el usuario.
  • Tipo y valor de contacto elegido por el usuario.
  • Estado de láminas o tarjetas: faltantes, obtenidas y repetidas.
  • Likes, dislikes, matches, búsquedas, bloqueos y reportes.
  • Límites diarios usados y actividad aproximada.

El contacto no se muestra públicamente. Solo puede ser visible para una persona específica cuando exista match mutuo y el usuario decida mostrar su contacto a esa persona.

3. Para qué usamos los datos

  • Crear y administrar cuentas.
  • Permitir que los usuarios registren su colección.
  • Calcular compatibilidad entre usuarios.
  • Mostrar posibles personas compatibles para intercambio.
  • Permitir likes, dislikes y matches.
  • Permitir que cada usuario decida a qué match mostrar su contacto.
  • Aplicar límites diarios de uso.
  • Prevenir spam, abuso, fraude o conductas inapropiadas.
  • Gestionar reportes y bloqueos.

4. Quién puede ver tus datos

Antes de un match, otros usuarios pueden ver información limitada, como nombre de usuario, comuna y cantidad de láminas o tarjetas compatibles.

Después de un match mutuo, otros usuarios pueden ver descripción y detalle de láminas compatibles.

El contacto no se muestra automáticamente a todos los matches. Cada usuario decide manualmente si quiere mostrar su contacto a una persona específica.

5. Servicios de terceros

La aplicación utiliza servicios de Firebase, incluyendo Firebase Authentication, Cloud Firestore y Firebase Hosting, para autenticación, base de datos y alojamiento.

Más adelante, la aplicación podría integrar servicios de anuncios, analítica o pagos. Si eso ocurre, esta Política de privacidad podrá ser actualizada.

6. Anuncios y funciones premium

La aplicación puede incluir usuarios gratuitos con límites diarios, espacios publicitarios, funciones para quitar anuncios y funciones premium.

7. Eliminación de cuenta

El usuario puede eliminar su cuenta desde la aplicación en: Mi perfil > Eliminar cuenta.

Al eliminar la cuenta, se eliminan los datos principales asociados al usuario, incluyendo perfil, colección, contacto privado, acciones propias, límites diarios, bloqueos y cuenta de acceso.

Algunos datos mínimos relacionados con seguridad, reportes o prevención de abuso podrían conservarse si es necesario para revisar incidentes, prevenir fraude, cumplir obligaciones legales o proteger a otros usuarios.

También puedes solicitar información o eliminación escribiendo a: trueque.gol.contacto@gmail.com

También puedes revisar las instrucciones públicas de eliminación de cuenta y datos en: Eliminar cuenta y datos.

8. Bloqueo y reportes

Los usuarios pueden bloquear o reportar a otros usuarios. Los reportes se usan para revisar casos de spam, estafa, perfil falso, contacto inapropiado u otros problemas.

9. Seguridad

Usamos Firebase y reglas de seguridad para limitar el acceso a la información. Sin embargo, ningún sistema es completamente infalible.

Recomendamos no compartir dirección exacta, datos sensibles ni información personal innecesaria en la descripción del perfil.

10. Menores de edad

La aplicación no está dirigida específicamente a niños pequeños. Si eres menor de edad, debes usar la aplicación con autorización y supervisión de tu madre, padre o tutor legal.

11. Ubicación

La aplicación no solicita ubicación exacta ni mapa en tiempo real. La compatibilidad se calcula usando ubicación aproximada seleccionada manualmente, como región y comuna.

12. Cambios a esta política

Podemos actualizar esta Política de privacidad cuando agreguemos nuevas funciones, anuncios, pagos, cambios legales o mejoras de seguridad.
