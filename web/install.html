Instalar TruequeGol como app

Puedes usar TruequeGol como si fuera una aplicación instalada, sin tener que entrar manualmente al navegador cada vez.

Android / Chrome

  1. Abre https://truequegol.cl en Chrome.
  2. Toca el menú de tres puntos ⋮.
  3. Selecciona Instalar app o Agregar a pantalla principal.
  4. Confirma tocando Instalar o Agregar.

iPhone / Safari

  1. Abre https://truequegol.cl en Safari.
  2. Toca el botón Compartir.
  3. Selecciona Agregar a pantalla de inicio.
  4. Confirma tocando Agregar.
Importante: en iPhone debes abrir la página desde Safari. En algunos navegadores la opción de instalación puede no aparecer.
